# STATIC_LAUNCH_MODE_REPORT.md

# Static Launch Mode — Call + WhatsApp v1

## Статус

Сайт переведён в режим первого статического запуска без CRM/backend.

Запуск допустим как:

- визитка;
- Local SEO landing для Сочи;
- каталог услуг;
- страница цен;
- калькулятор предварительной стоимости;
- приём реальных обращений через звонок и WhatsApp.

## Что изменено в CTA

- Основные CTA на ключевых страницах переведены на:
  - `Позвонить`;
  - `Написать в WhatsApp`;
  - `Рассчитать стоимость`.
- CTA `Оставить заявку` убраны из приоритетных мест.
- На `contacts.html` форма явно обозначена как тестовая.
- `thanks.html` больше не обещает реальную отправку заявки или связь менеджера.
- Telegram оставлен только как placeholder в footer/контактах и не используется как основной канал.

## Как работает WhatsApp

- Базовый номер: `https://wa.me/79011598267`.
- Общий текст сообщения хранится в `assets/js/site-config.js` как `contacts.whatsappDefaultText`.
- На service-страницах используются контекстные сообщения через `data-whatsapp-message`.
- `assets/js/main.js` формирует WhatsApp URL с `?text=...` и безопасно кодирует сообщение через `encodeURIComponent`.
- Если JavaScript недоступен, базовые WhatsApp-ссылки в HTML уже ведут на `wa.me/79011598267` с общим текстом.

## Калькулятор

- После расчёта показывается предварительная стоимость.
- Пояснение подчёркивает, что цена уточняется после оценки материала, состояния и адреса выезда.
- Кнопка `Отправить расчёт в WhatsApp` формирует сообщение с:
  - выбранной услугой;
  - площадью или количеством;
  - дополнительными опциями;
  - предварительной ценой;
  - городом Сочи.
- Данные не отправляются автоматически: WhatsApp открывается только по клику пользователя.

## Формы

- Формы остаются в mock/demo-режиме.
- На `contacts.html` добавлено честное предупреждение: для быстрой реальной заявки нужно звонить или писать в WhatsApp.
- Demo-форма не отправляет данные во внешние системы.
- После успешной demo-проверки открывается `thanks.html`.
- Персональные данные не пишутся в `console.log`.

## Можно ли запускать как визитку

Да, сайт можно запускать как статическую визитку и Local SEO landing, если владелец принимает ограничения:

- реальные заявки идут через телефон и WhatsApp;
- CRM/backend пока отсутствуют;
- форма работает только как demo;
- privacy остаётся черновиком до юридической проверки.

## Что блокирует полноценный production

- Финальная политика обработки персональных данных.
- Уточнение юридического оператора персональных данных.
- Backend/serverless endpoint для реальных форм.
- Карта/ссылка на профиль организации.
- Telegram username/link, если канал нужен.
- Реальные фото работ.
- Минимальная сумма заказа.
- Финальная проверка цен владельцем.

## Проверено

- CTA на ключевых страницах не обещают реальную CRM-отправку.
- WhatsApp-ссылки ведут на `wa.me/79011598267`.
- Телефонные ссылки используют `tel:+79011598267`.
- Telegram не ведёт на фейковую рабочую ссылку.
- Калькулятор формирует WhatsApp-сообщение с расчётом.
- Форма остаётся mock/demo.
- `thanks.html` сообщает о demo-режиме.
