## 6. Статус sitemap.xml / robots.txt

### sitemap.xml
- **Статус:** требует обновления после получения домена
- **Текущее состояние:** все URL относительные (`index.html`, `services.html`, …) — невалидно по протоколу sitemap.org
- **Действие при получении домена:** заменить каждый `<loc>` на `https://домен.ru/страница.html`; исключить `thanks.html` и `privacy.html` из индексации (добавить `<changefreq>` и `noindex`-подсказку) по желанию

### robots.txt
- **Статус:** требует обновления после получения домена
- **Текущее состояние:** `Sitemap: sitemap.xml` — относительный путь, не валиден для Sitemap-директивы
- **Индексация:** не блокируется (`User-agent: * Allow: /`) — OK
- **Действие при получении домена:** заменить на `Sitemap: https://домен.ru/sitemap.xml`

### og:image
- Все страницы: `content="assets/img/hero-cleaning.jpg"` — относительный путь, не будет работать в превью соцсетей
- **Действие при получении домена:** заменить на `https://домен.ru/assets/img/hero-cleaning.jpg` во всех 14 HTML-файлах
